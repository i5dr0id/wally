A simple wallpaper changer for gnome ;)

Pre-requirements
==================
 * Linux, any distribution runing GNOME enviroment.

 * NODEJS.

## Usage
==================
```shell
node wally.js 60 $PATH_TO_IMAGE_DIRECTORY/
```
        OR

```shell
./wally.js 60 $PATH_TO_IMAGE_DIRECTORY/
```

60 represent 60 seconds
